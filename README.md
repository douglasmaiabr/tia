# Tia 

Sistema integrado para gerenciamento de desempenho escolar.

## Dados
+ Escola
* Nome da escola
* Endereço
* Telefone
* E-mail

+ Professores
* Nome
* Telefone
* E-mail
* Graduação

+ Alunos
* Nome
* Endereço
* Filiação
* Data de nascimento
* Foto
* Data de matrícula

+ Turmas

+ Disciplinas
